    # Elento Brent – Personal Webpage Project

## Project Overview
This project is a personal webpage created using **HTML5** and **CSS3**. The webpage demonstrates foundational web design concepts including semantic HTML structure, Flexbox layout, responsive design, tables, forms, and CSS transitions.

The site contains multiple sections such as About, Projects, and Contact, and follows accessibility and usability best practices.

---

## Features
- Semantic HTML elements (`header`, `nav`, `main`, `section`, `footer`)
- Navigation menu using Flexbox
- Responsive layout for phone, tablet, and desktop screens
- Projects section displayed using an HTML table
- Contact form with labeled inputs and required fields
- CSS hover, focus, and transition effects
- External stylesheet for clean separation of structure and style

## How to View the Project
1. Download or clone the project files.
2. Make sure all files are located in the same root directory.
3. Open `index.html` in a web browser.

---

## Accessibility Considerations
- All form inputs are paired with `<label>` elements
- Images include descriptive `alt` text
- Clear heading hierarchy is used throughout the page
